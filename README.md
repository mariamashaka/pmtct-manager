# pmtct-manager
# PMTCT Management System / Mfumo wa Usimamizi wa PMTCT

## English

### Description
A comprehensive web-based management system for Prevention of Mother-to-Child Transmission (PMTCT) of HIV programs. This system helps healthcare workers track patients, manage appointments, monitor viral loads and CD4 counts, and generate reports for ministry requirements.

### Features
- **Bilingual Interface**: Switch between English and Swahili
- **Patient Management**: Track mothers and children in PMTCT program
- **Automated Alerts**: Notifications for missed visits, overdue tests, and critical results
- **Real-time Statistics**: Dashboard with key metrics (VL suppression, CD4 counts, etc.)
- **Medication Tracking**: Monitor ARV adherence, co-trimoxazole, IPT, and additional medications
- **Child Testing Schedule**: Automated reminders for DBS and Bioline tests
- **Monthly Reports**: Generate cohort reports for ministry submission
- **Offline Capability**: Works without internet, stores data locally
- **Data Export/Import**: Backup and transfer data between devices

### Technology
- Pure HTML/CSS/JavaScript (no frameworks required)
- Tailwind CSS for styling
- localStorage for data persistence
- No server required - runs completely in browser

### Installation
1. Download all files
2. Open `index.html` in any modern web browser
3. No installation or setup required!

### Usage
1. **Add Patients**: Register mothers and their children
2. **Record Visits**: Document each visit with tests and medications
3. **Monitor Alerts**: Review automated alerts for patients needing attention
4. **Generate Reports**: Create monthly cohort reports for ministry

### Data Storage
All data is stored locally in your browser using localStorage. Remember to:
- Regularly export backups
- Don't clear browser data without backing up first
- Use the Export/Import feature to transfer data between devices

## Kiswahili

### Maelezo
Mfumo kamili wa wavuti kwa usimamizi wa programu za Kuzuia Maambukizi ya VVU kutoka kwa Mama kwenda kwa Mtoto (PMTCT). Mfumo huu unasaidia wahudumu wa afya kufuatilia wagonjwa, kusimamia miadi, kufuatilia virusi na hesabu ya CD4, na kutengeneza ripoti kwa mahitaji ya wizara.

### Vipengele
- **Lugha Mbili**: Badilisha kati ya Kiingereza na Kiswahili
- **Usimamizi wa Wagonjwa**: Fuatilia mama na watoto katika programu ya PMTCT
- **Tahadhari Otomatiki**: Arifa za ziara zilizokosekana, vipimo vilivyochelewa, na matokeo muhimu
- **Takwimu za Moja kwa Moja**: Dashibodi yenye vipimo muhimu (VL, CD4, n.k.)
- **Ufuatiliaji wa Dawa**: Fuatilia ufuatiliaji wa ARV, co-trimoxazole, IPT, na dawa nyingine
- **Ratiba ya Vipimo vya Watoto**: Vikumbusho otomatiki kwa vipimo vya DBS na Bioline
- **Ripoti za Kila Mwezi**: Tengeneza ripoti za kohoti kwa kuwasilisha wizara
- **Uwezo wa Nje ya Mtandao**: Inafanya kazi bila intaneti, inahifadhi data ndani
- **Kuhamisha Data**: Hifadhi na uhamishie data kati ya vifaa

### Teknolojia
- HTML/CSS/JavaScript safi (hakuna frameworks zinazohitajika)
- Tailwind CSS kwa mitindo
- localStorage kwa uhifadhi wa data
- Hakuna seva inayohitajika - inafanya kazi kabisa kwenye kivinjari

### Usakinishaji
1. Pakua faili zote
2. Fungua `index.html` kwenye kivinjari chochote cha kisasa
3. Hakuna usakinishaji au usanidi unaohitajika!

### Matumizi
1. **Ongeza Wagonjwa**: Sajili mama na watoto wao
2. **Rekodi Ziara**: Andika kila ziara na vipimo na dawa
3. **Angalia Tahadhari**: Kagua tahadhari otomatiki kwa wagonjwa wanaohitaji umakini
4. **Tengeneza Ripoti**: Unda ripoti za kohoti za kila mwezi kwa wizara

### Uhifadhi wa Data
Data yote inahifadhiwa ndani kwenye kivinjari chako kwa kutumia localStorage. Kumbuka:
- Hamisha nakala za hifadhi mara kwa mara
- Usifute data ya kivinjari bila kuhifadhi kwanza
- Tumia kipengele cha Kuhamisha/Kupakia ili kuhamisha data kati ya vifaa
